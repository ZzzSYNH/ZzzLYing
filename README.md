# 基于STM32F103C8T6的智能手表设计
小组成员：24微电4班 张俪瀛，张峻浩，温深航
## 一.介绍
该智能手表项目是一个基于STM32F103C8T6的嵌入式系统，采用FreeRTOS实时操作系统实现多任务管理，集成OLED显示和陀螺仪姿态检测等功能。项目涵盖硬件电路设计、RTOS任务调度、I2C总线通信及低功耗电源管理，具有较好的学习及实践价值。
